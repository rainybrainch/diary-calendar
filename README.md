# 📔 Forest Note - 日記カレンダー

毎日の日記を記録し、AI が自動整理する習慣追跡アプリ。**デモモード完全対応・手動セットアップ不要**。

🌐 **本番デモ**: https://diary-calendar-mu.vercel.app  
📦 **バージョン**: v1.2.0 (AI統合版)  
🚀 **ステータス**: 安定版・本番運用可能

---

## 🎯 コンセプト

朝の日常利用フローを想定した、**AI伴走型習慣追跡日記アプリ**。

- **毎朝5分の記録習慣** → テキストペースト → AI質問チャットで深掘り
- **AI自動整理** → タイトル・タグ・要約を自動生成
- **習慣達成度をビジュアル化** → カレンダーのヒートマップ
- **カード収集の喜び** → AIタグ・属性をカードに反映 → コレクション欲求
- **ランキング参加** → 友人と競う → 継続動機付け

## ✨ 実装済み機能

### 🏠 ホーム画面
- **🎯 今日の習慣** - 6つのチェックボックス（pushups/squats/plank/run/reading/ai_learning）
- **📊 統計表示** - 連続日数・今日の達成数・気分スコア
- **📅 カレンダー** - 習慣達成度のヒートマップ表示（6段階色分け）
- **ボトムナビ** - 4つの主要機能へのワンタップアクセス

### ✍️ Forest Note 入力画面
- **テキストペースト** - Ctrl+V で貼り付け
- **リアルタイムプレビュー** - 入力中に解析結果を表示
- **自動解析** - FOREST_NOTE形式のテキストを自動パース（MENTAL/BODY/WORK/RELATION/MONEY/HABIT/DREAM）

### 💭 AI 質問チャット
- **5段階の質問** - 気分・イベント・学び・体調・目標
- **フォローアップ質問** - 最大3回までの深掘り質問
- **進捗表示** - プログレスバーで進捗可視化
- **スキップ可能** - 不要な質問はスキップ
- **Demo AI** - API キー不要（固定ロジック、即座に動作）

### ✨ AI 要約生成
- **タイトル自動生成** - 日付 + キーワード（編集可）
- **要約文自動生成** - 質問の回答から要約（編集可）
- **タグ自動生成** - 3～8個の関連タグ（追加・削除可）
- **属性選択** - 7種類（mind/body/work/relation/money/habit/dream）
- **気分・エネルギー設定** - 1～5段階スケール
- **編集UI** - すべてのフィールドをワンページで編集可能

### 🎴 AI カード統合
- **AI タイトルをカード表示** - 生成されたタイトルをカードに反映
- **AI タグをカード表示** - タグがある場合は上位3個をカード表裏に表示
- **AI 属性をカード属性に反映** - カードの色・デザインに反映
- **AI 気分・エネルギーを統計に反映** - カード統計値に反映

### 🎴 カード一覧画面
- **グリッド表示** - レスポンシブ対応（2～4列自動調整）
- **レアリティフィルター** - common/rare/epic/legendary別表示
- **ソート機能** - 新しい順 / レアリティ順
- **進捗バー** - 月間達成率を視覚化
- **フリップアニメーション** - カード表裏の3D回転

### 🏆 ランキング画面
- **表彰台デザイン** - 1～3位を金・銀・銅で表示
- **自分の順位固定表示** - sticky で常時表示（3位以下）
- **タブ切り替え** - 今日・週間・月間・連続達成
- **メダル表示** - 順位を視覚的に強調

### 📅 カレンダー機能
- **習慣達成度ヒートマップ** - 6段階の色分け（緑→オレンジ）
- **月の統計** - 記録数・達成率・プログレスバー
- **日付インジケータ** - 気分・体力の絵文字表示
- **前月・次月移動** - スムーズな月間閲覧

### 🔐 デモモード
- **localStorage基盤** - Supabase不要で即座に動作
- **自動サンプルデータ** - 初回アクセス時に14日分を生成
- **手動セットアップ不要** - クリック1つでデモ開始

## 📚 AI 機能の使い方

### 典型的なユーザーフロー

```
1️⃣ テキスト入力 (/input/paste)
   ↓
   テキストを貼り付け → Forest Note 形式を自動解析
   
2️⃣ 💭 AI 質問チャット (/input/ai-questions)
   ↓
   5 つの質問に答える（気分・イベント・学び・体調・目標）
   各質問に 1～2 分程度
   
3️⃣ ✨ AI 要約生成 (/input/ai-summary)
   ↓
   AI が自動生成：タイトル・要約・タグ・属性
   気分・エネルギーを設定
   すべて編集可能
   
4️⃣ 🎴 カード確認 (/input/confirm)
   ↓
   AI 生成コンテンツが反映されたカードを確認
   カード表裏はフリップしてプレビュー
```

### Demo AI の特性

**初期状態（デフォルト）**
- API キー不要
- 固定ロジックで質問・タイトル・タグを生成
- 本番環境でそのまま動作
- 応答が即座（遅延なし）

**想定される出力例**
```
質問: 今日の朝、気分はいかがでしたか？ (1-5 段階)
答え: 4

AI 生成タイトル: "07 (月) - 今日の成長"
AI 生成タグ: ["日常", "学び", "成長", "感動"]
AI 属性: "mind" (メンタル)
```

## 🔧 デモモード ↔ 本番モード

### デモモード（デフォルト）
```bash
# .env.local
NEXT_PUBLIC_DEMO_MODE=true
NEXT_PUBLIC_SUPABASE_URL=https://placeholder.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=placeholder
NEXT_PUBLIC_AI_PROVIDER=demo
```

**特性**:
- ✅ Supabase不要
- ✅ localStorage にデータ保存
- ✅ 初回自動でサンプルデータ生成
- ✅ 仮ユーザーで自動ログイン
- ✅ Demo AI で即座に動作（API キー不要）

### 本番モード（Supabase有効 + 任意のAI）
```bash
# .env.local

# Supabase 設定
NEXT_PUBLIC_DEMO_MODE=false
NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key

# AI Provider 設定（オプション）
NEXT_PUBLIC_AI_PROVIDER=demo              # "demo" | "openai" | "gemini"
NEXT_PUBLIC_AI_API_KEY=sk-...            # API キー（provider が "demo" の場合は不要）
NEXT_PUBLIC_AI_MODEL=gpt-3.5-turbo       # モデル名
```

**AI Provider の選択**:
- `demo` - 固定ロジック（推奨・API キー不要）
- `openai` - OpenAI API（v1.3 以降で実装予定）
- `gemini` - Google Gemini API（v1.3 以降で実装予定）

Supabase セットアップは [docs/SUPABASE_SETUP.md](./docs/SUPABASE_SETUP.md) を参照。
