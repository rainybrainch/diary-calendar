# 🎴 カード進化システム仕様書

## 概要
毎日の日記記録が「ゲームカード」として進化。Lv 1-100 で段階的にレアリティが上昇し、心理的な成長を実感させます。

---

## EXP 計算式

### 毎日獲得する EXP
```
Daily EXP = 習慣EXP + ストリークEXP + 気分EXP + エネルギーEXP
最大値: 50 EXP/日
最小値: 1 EXP/日
```

### 詳細計算

| 要因 | 計算 | 上限 |
|-----|------|------|
| 習慣達成 | 達成数 × 3 | 18 |
| ストリーク | 3日:5, 7日:10, 14日:15, 30日:20 | 20 |
| 気分（4-5） | 5 | 5 |
| エネルギー（4-5） | 5 | 5 |
| **合計** | | **50** |

### 例
```
習慣6個達成 + 連続30日 + 気分5 + エネルギー5
= 18 + 20 + 5 + 5 = 48 EXP
```

## データ構造

```typescript
interface CardEvolutionState {
  level: number;           // 1-100
  exp: number;             // 累積 EXP
  nextLevelExp: number;    // 次レベルまでの必要EXP
  currentLevelExp: number; // 現レベルでの現在EXP
  rarity: CardRarity;      // 1-5 (Normal/Rare/Epic/Legendary/Mythic)
  growthRate: number;      // 成長率（0-100%）
  evolvedAt?: string;      // 進化日時（ISO）
  isNewEvolution?: boolean;// 今日進化したか
}
```

## 長期モチベーション設計

### 3つのマイルストーン

1. **短期（1-2週間）**: Rare 到達 → 「最初の成功」
2. **中期（1-2ヶ月）**: Epic 到達 → 「習慣定着」
3. **長期（3-6ヶ月）**: Legendary 到達 → 「自分への信頼」

最終目標 Mythic（1年）は「終わりのない継続」を象徴します。
