# 🌲 森システム仕様書

## 概要
ユーザーの習慣達成・連続記録・カード数に基づいて、リアルタイムで「森」が成長するビジュアルシステム。ユーザーの庭に樹木が育つことで、心理的な達成感と継続の動機付けを実現します。

---

## 成長メカニズム

### 計算式
```
ForestLevel = (習慣達成数 × 5) + (連続記録日数 × 2) + (総カード数 × 1)
最大: Lv 100
```

### 段階的な解放

| Level | 変化 | トリガー |
|------|------|---------|
| 0-9 | 背景のみ | 初期状態 |
| 10-19 | 🌱 草が生える | 最初の変化 |
| 20-29 | 🌿 草が繁茂 | 継続が増す |
| 30-39 | 🌳 樹木1本 | 視覚的な達成感 |
| 40-49 | 樹木が大きく | 成長実感 |
| 50-59 | 🌳🌳 樹木2本 | 中盤到達 |
| 60-69 | 樹木がさらに大きく | 大規模化 |
| 70-79 | 🌳🌳🌳 樹木3本 | クライマックス |
| 80-89 | 動物登場（🦋 蝶・🐦 鳥） | 生態系の形成 |
| 90-99 | 季節変化（🍂 秋・❄️ 冬） | 循環の実感 |
| 100 | 🌲 完全な森 | 最高地点 |

### 季節変化（Level 90+）
- **春（Lv 0-49）**: 明るい緑・🌸 花
- **夏（Lv 50-79）**: 深い緑・☀️ 太陽
- **秋（Lv 80-94）**: 🍂 紅葉・🦇 コウモリ
- **冬（Lv 95-100）**: ⛄ 雪・❄️ 氷

### 動物の登場
- **Lv 80+**: 🦋 蝶（春）
- **Lv 85+**: 🐦 鳥（夏）
- **Lv 90+**: 🐰 ウサギ（秋）
- **Lv 95+**: 🦌 鹿（冬）
- **Lv 100**: 🦊 キツネ（最終）

### コンポーネント構成
```
<svg viewBox="0 0 400 300">
  <!-- 背景グラデーション -->
  <defs>
    <linearGradient id="sky" x1="0%" y1="0%" x2="0%" y2="100%">

      <stop offset="100%" style="stop-color:rgb(135,206,250)"/>

  {level >= 50 && <Tree2 x="200" />}
  {level >= 70 && <Tree3 x="300" />}
  
  <!-- 花 -->
  {level >= 40 && <Flowers />}
  
  <!-- 動物 -->
  {level >= 80 && <Butterfly />}
  {level >= 85 && <Bird />}
  ...
</svg>
```

### アニメーション
- **成長**: 樹木が徐々に大きくなる（Grow animation）
- **新要素追加時**: フェードイン + スケール
- **動物**: ゆっくりした浮遊アニメーション

## ユーザー体験

### 初期段階（Lv 0-29）
- 「今は何もない」状態
- ユーザーに焦点：「自分の行動で森が育つ」という希望を提供

### 成長段階（Lv 30-69）
- 視覚的な変化が明確
- 樹木の本数増加で達成感
- 「森が育っている」実感

### 発展段階（Lv 70-99）
- 動物・季節変化で複雑さ増加
- 「大規模な生態系」の形成
- 継続への心理的報酬

### 完成段階（Lv 100）
- 🌲 最高の森ビジュアル
- 実績ページで特別な表示
- 永遠の継続へのシンボル
